# starter-laravel-vue-spa

## 注意事項

1. 不採用 `laravel/ui` 套件
2. bootstrap / vue or react 不透過 php artisan ui 管理，直接使用 npm / yarn 獨立管理
3. 專案根目錄 `/` 為標準 laravel project 資料夾結構
4. 專案目錄 `/webapp-web` 為 nodejs / webpack 專案目錄

---

## Install
### laravel
php packages
```shell
composer install
```

### nodejs  (**optional**)
nodejs packages
```shell
yarn install
```
or 
```shell
npm install
```

## Development
### laravel
run with

```shell
php artisan serve
```

open `http://127.0.0.1:8000`
